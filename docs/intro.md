## Module Overview

### Module 1: ROS 2 Nervous System
Learn ROS 2 fundamentals, Python packages, and URDF creation for humanoid robots.

### Module 2: Digital Twin (Gazebo & Unity)
Master physics simulation, sensors, and visualization techniques.

### Module 3: AI-Robot Brain (NVIDIA Isaac)
Explore Isaac Sim, synthetic data generation, and perception pipelines.

### Module 4: Vision-Language-Action
Integrate voice input, LLM planning, and action execution.

### Module 5: Capstone - Autonomous Humanoid
Build an end-to-end system that brings everything together.

## Prerequisites

- **Hardware**: Mid-range desktop with 16GB RAM, RTX 3060, 6-core CPU
- **Software**: Ubuntu 22.04, ROS 2 Humble, Python 3.10+
- **Knowledge**: Intermediate programming skills, basic robotics concepts
