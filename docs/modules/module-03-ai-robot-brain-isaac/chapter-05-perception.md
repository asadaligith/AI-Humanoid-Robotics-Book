# Chapter 5: Object Pose Estimation

## Learning Objectives

1. **Run** DOPE (Deep Object Pose Estimation) for 6D pose detection
2. **Train** custom object models
3. **Integrate** pose estimation with manipulation planning

## DOPE Overview

DOPE predicts 6D object poses (position + orientation) from RGB images using deep learning.

**Launch**:
```bash
ros2 launch isaac_ros_dope isaac_ros_dope.launch.py
```
